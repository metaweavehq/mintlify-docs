## What it does

The **ME Performance Microapp Collector** captures a complete ME performance evidence package by combining microapp API data with browser screenshots of every dashboard tab. The result includes not only current performance values but also the baseline polynomial curves and shop-trial overlays needed for proper deviation analysis.

It is an alternative to the standard [ME Performance Data Collector](/skills/me-performance-data-collector), used when the vessel's ME data lives in the ME Performance microapp rather than the ERP.

## When to use it

- "Microapp ME data"
- "Performance screenshots"
- "Fetch ME from microapp"
- "Evidence from the ME dashboard"

## What it captures

| Data | Description |
|------|-------------|
| Performance records | Every recorded performance entry for the vessel |
| Baseline curves | Polynomial baseline used for deviation analysis |
| Shop-trial data | Original shop-trial values for overlay comparisons |
| Tab screenshots | Full-page captures of each microapp tab — overview, deviation, trends, etc. |

## How it works

1. Checks browser tooling availability — needed for screenshots.
2. Pulls the three API endpoints (performance, baseline, shop-trial) for the configured vessel.
3. Drives a headless browser through every microapp tab, capturing a full-page screenshot of each.
4. Bundles structured data + screenshots into a single timestamped evidence record under the vessel's ME case folder.
5. Generates a plain-English summary covering record count, baseline coverage, latest deviation flags.

If browser tooling is unavailable the API data still produces a valid evidence package — visuals are simply skipped.

## What it produces

- Structured performance, baseline, and shop-trial data
- Tab-by-tab dashboard screenshots
- Plain-English summary
- Filing metadata

## Why visuals matter

Engine reviews frequently end up in front of charterers, owners, or class — anyone reading a deviation claim wants to see the same screen the operator saw. Pairing API data with screenshots makes the evidence defensible.
