# Document Search

Document Search provides AI-powered search and retrieval across your organization's knowledge base.

## What is document search?

Document Search uses semantic understanding to find relevant information across documents, databases, and knowledge repositories — going beyond simple keyword matching.

## Key capabilities

- **Semantic search** — Understand intent and meaning, not just keywords
- **Multi-source** — Search across documents, databases, wikis, and more
- **Contextual answers** — Get direct answers with source citations
- **Access control** — Respect existing permissions and security policies
