# Microapps

Microapps are lightweight, purpose-built applications that combine AI agents with user interfaces to solve specific business problems.

## What are microapps?

Microapps bridge the gap between raw AI capabilities and end-user experiences. They wrap agent workflows in intuitive interfaces that anyone can use — no AI expertise required.

## Key capabilities

- **Purpose-built** — Each microapp solves a specific problem
- **AI-powered** — Backed by agents and skills for intelligent automation
- **User-friendly** — Simple interfaces accessible to non-technical users
- **Rapid deployment** — Build and deploy in hours, not weeks
